# Weather Forecast Application

This weather forecast application provides users with the ability to check weather conditions and forecasts for various locations. It is built using Python for the backend and HTML for the frontend.

## Technologies Used
- **Python** (88%): Responsible for fetching weather data, processing user requests, and managing application logic.
- **HTML** (12%): Utilized to create the user interface, allowing users to interact with the application.

## Features
- Current weather data retrieval.
- 5-day weather forecast.
- User-friendly interface for easy navigation.

## Installation
To run this application on your local machine, first clone the repository, then install the necessary dependencies using:
```bash
pip install -r requirements.txt
```

## Usage
Run the application by executing:
```bash
python app.py
```
Open your web browser and navigate to `http://localhost:5000` to see the app in action!
